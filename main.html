2.What is the difference between innerHTML, innerText, and textContent ?

Use innerHTML when you need HTML formatting.
Use innerText when you want visible text only.
Use textContent when you want raw text quickly.

3.What is event delegation in the DOM?

Event delegation is a technique in JavaScript where you attach a single event listener to a parent element instead of adding separate listeners to multiple child elements. Then, inside the event handler, you check which child element triggered the event and handle it accordingly.

4. What is event bubbling in the DOM?

vent bubbling is a behavior in the DOM where an event triggered on a child element “bubbles up” through its ancestors, triggering event listeners on parent elements as well.

5. How do you create, add, and remove elements using JavaScript?

createElement() only creates the element in memory; it’s not in the DOM until appended. appendChild() always adds at the end, insertBefore() lets you choose the position. remove() is simpler than removeChild(), but removeChild() is useful if you only have access to the parent.
